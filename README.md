# pdd-187 智能家居设备状态

智能家居设备状态可视化页面，展示各房间设备运行状态与环境数据。

## 功能

- 顶部房间切换标签（客厅、卧室、厨房、书房）
- 设备卡片展示：灯（开关+亮度滑条）、空调（温度+模式）、摄像头（缩略图+录制状态）、窗帘、净化器、音箱
- 环境总览：温度、湿度、空气质量指数仪表盘
- 能耗统计：今日用电量折线图 + 各设备用电占比饼图
- 设备离线时卡片变灰并提示
- 支持开关控制、亮度调节、温度调节、模式切换，操作后有反馈动画
- 数据每 5 秒自动刷新

## 技术栈

- React 18 + TypeScript + Vite
- TailwindCSS
- ECharts + echarts-for-react
- Lucide React 图标

## 运行

```bash
cd pdd-187
npm install
npm run dev
```

默认端口 3000。
